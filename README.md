## Description

Solana Programs Monitor

## Getting Started

### Dependencies

- [Node.js](https://nodejs.org/) (v18 or later)
- [TypeScript](https://www.typescriptlang.org/download/) (latest version)

### Installing

```shell
npm install
```

### Executing program
e.g. monitoring pumpFun migration

```shell
ts-node pumpFunMigration.ts
```
